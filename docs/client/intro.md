# Client Intro

**Horizon Truth Client** is the modern web interface for the Horizon Truth platform. It provides an intuitive, responsive, and robust experience for users to interact with claims, reviews, and platform data.

## Features
- **Modern UI**: Built with React 19 and styled using Tailwind CSS v4.
- **Fast Build Times**: Powered by Vite.
- **Strict Typing**: Full TypeScript support for reliability.
- **State Management**: Zustand for global state management.
- **Data Fetching**: React Query for efficient and cached API interactions.

## Tech Stack
- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 7](https://vitejs.dev/)
- **Language**: TypeScript 5.9
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **State Management**: [Zustand](https://zustand-demo.pmnd.rs/)
- **Data Fetching**: [React Query](https://tanstack.com/query/latest)
- **Testing**: Vitest & React Testing Library
- **UI Components**: Radix UI primitives & Lucide React icons
