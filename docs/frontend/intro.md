# Frontend Intro

**Horizon Truth Client (v2)** is the modern web interface for the Horizon Truth platform. It provides an intuitive and responsive experience for users to interact with claims, reviews, and platform data.

## Features
- **Modern UI**: Built with React 19 and Tailwind CSS 4.
- **Fast Build Times**: Powered by Vite.
- **Strict Typing**: Full TypeScript support for reliability.
- **Real-time Interaction**: Seamless integration with the Horizon Truth API.

## Tech Stack
- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 7.2](https://vitejs.dev/)
- **Language**: TypeScript 5.9
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Testing**: Vitest & React Testing Library
