# Frontend Documentation - horizon-truth-client-v2

Welcome to the documentation for the **Horizon Truth Frontend**.

## Overview
This project is built using React and Vite, utilizing Tailwind CSS for styling.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
```bash
cd horizon-truth-client-v2
npm install
```

### Running Locally
```bash
npm run dev
```

## Project Structure
- `src/components`: Reusable UI components.
- `src/pages`: Main application pages.
- `src/stores`: State management logic.
